SOBRE

O nosso projeto foi desenvolvido para um trabalho da disciplina de Desenvolvimento Web, ministrada pela Professora Danielle Costa de Oliveira, onde exigia a criação de um sistema Web.
A decisão foi a criação de um sistema bancário, com uso de Html, CSS e Javascript, em que o sistema forneceria a menor combinação de notas para o usuário.

SAQUE

O nosso sistema disponibiliza uma eficiente contagem de notas, que facilita o usuário andar com o menor número de cédulas possível
